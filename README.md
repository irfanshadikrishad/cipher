#### @irfanshadikrishad/cipher

Minimalist Cryptography Implementaion

#### Usage

```js
import { Cipher } from "@irfanshadikrishad/cipher"

const ceaser = new Cipher.Ceaser(6)

console.log(ceaser.encrypt("abx"))
```

#### Available Ciphers

- Ceaser Cipher
- Atbash Cipher
- Playfair Cipher
- Vigenere Cipher
